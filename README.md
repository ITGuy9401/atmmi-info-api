# site-2-api

Just a small test in creating a service in Node.js that turns a website in a REST API.

## Built with
* Node.js 8.2.1
* [Hapi](https://hapijs.com/)
* [X-ray](https://github.com/matthewmueller/x-ray)

## Install & run
Clone and run 
```
$ yarn
```

starts the service
```
$ yarn start
```

as above but with Nodemon
```
$ yarn dev
```
